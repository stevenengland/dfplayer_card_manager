# DfPlayer (Mini) Card Manager

This package is intended to manage SD cards for their usage in DFROBOTs DFPlayer SD card reader.

# Features

- Clean SD card
- Check SD card for common errors
- Apply FAT sort

## What is not covered?

- SD card formatting
